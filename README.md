🧮 Python Calculator
A simple calculator built using Python that performs basic arithmetic operations such as addition, subtraction, multiplication, division and Module.
Features:
User-friendly interface
Handles basic math operations
Error handling for invalid inputs
Lightweight and beginner-friendly project
Technologies Used:
Python
How It Works:
The user inputs two numbers.
Selects an operation (Add, Subtract, Multiply, Divide and Module).
The calculator displays the result instantly.
Future Improvements:
Add GUI using Tkinter
Add advanced operations (square root, power, etc.)
